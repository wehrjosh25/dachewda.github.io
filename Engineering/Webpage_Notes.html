How to Upload an Extension to Firefox

    Load it into Firefox Temporary Add-ons

  1. Go to: about:debugging#/runtime/this-firefox
  2. Click the “Load Temporary Add-on…” option.
  3. Select the manifest.json file of your website extension.

    4. Firefox will load the extension and show its icon in the toolbar.
